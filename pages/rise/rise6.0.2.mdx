Download Here : [link](https://sub2unlock.io/eRPSu)

1. Open notepad as admin
2. Go to C:\Windows\System32\drivers\etc
3. Select option to open all files with notepad
4. Open hosts file
5. Add 127.0.0.1 auth.riseclient.com to hosts
6. Save
7. Put Rise in .minecraft\versions
8. Put Libraries in .minecraft/libraries
9. Run start.bat (don't close it)
10. Run Rise in your mc launcher
11. Enjoy

If Rise don't load use jre1.8.0_51 and these arguments:
```bash
-noverify -Xmx2G -XX:+UnlockExperimentalVMOptions -XX:+UseG1GC -XX:G1NewSizePercent=20 -XX:G1ReservePercent=20 -XX:MaxGCPauseMillis=50 -XX:G1HeapRegionSize=32M
```
